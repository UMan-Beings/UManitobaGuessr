title Accounts

participant Frontend
participant Backend
participant Database

entryspacing 0.7
==Sign Up (Successful)==
Frontend->Backend:POST /api/v1/signup {email, username, password}
Backend->Database:Email Registered?
Backend<-Database:No
Backend->Database:Email, Username, Password Hash
Frontend<-Backend:Sign Up Success
==Sign Up (User Already Exists)==
Frontend->Backend:POST /api/v1/signup {email, username, password}
Backend->Database:Email Registered?
Backend<-Database:Yes
Frontend<-Backend:Account Already Exists
==Sign In==
Frontend->Backend:POST /api/v1/signin {username, password}
Backend->Database:Username, Password Hash
Backend<-Database:Valid
Frontend<-Backend:Sign In Confirmation
